# Nod - Agent Instructions & Architecture Context

This `AGENTS.md` file serves as the definitive architecture and context guide for any future AI agents or contributors working on the **nod** repository.

## 🌟 Project Overview

**nod** is a lightweight, cross-platform endpoint probe that monitors real-time device activity. It silently tracks input and screen lock states to verify true user presence, built with a strict **local-first** and **serverless** mindset.
